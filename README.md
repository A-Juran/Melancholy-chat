## Melancholy-chat（微聊天室）
为了使个人业务能力及开发能力得到提升，所以在百忙之中决定抽出空余时间进行此项目的开发。
### 技术栈
前端：SpringBoot + MybatisPlus + SpringWebSocket + redis + JWT
前端：UniApp + Uni-UI
